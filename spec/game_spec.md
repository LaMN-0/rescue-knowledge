# 救急・救命・救助知識学習RPG 仕様書（全体概要）

## 1. 前提条件とコンセプト

### 1.1 「プレイヤーの役に立つ」定義の適合チェック
本ゲームは、プレイすることで以下の価値をプレイヤーに提供することを目指します。
* **知識の増加**: 救急、救命、救助に関する実用的な知識（心肺蘇生法、応急手当、災害時の避難行動など）をゲーム体験を通じて学ぶ。
* **スキルの向上**: 学んだ知識をシミュレーション（ゲーム内イベント・バトル）で繰り返し実践し、現実世界での判断力や行動力を高める。
* **問題の解決・緩和**: 日常生活や非常事態において、パニックに陥らず適切な一次対処ができるようになる。

> **【前提条件との適合判定】**
> * **現状**: 救急・救命・救助のRPGというコンセプトは、前提条件に完全に適合しています。
> * **エンタメ（ファンタジー）世界観への適合**: 世界観はファンタジー（魔道具等が登場するエンタメ志向）ですが、作中で用いる「救命技術」「救助手順」「応急処置」は**現実の正しい知識に完全準拠**します。名称や演出のみを魔道具風（例：AED→古代の蘇生魔石など）に翻訳してエンタメ性を高めつつ、手順や救護ロジックを正確に再現することで、「現実世界で役立つ正しい知識」が身につく仕様とします。

## 3. 基本ゲームサイクル

### 3.1 進行構造（ステージ選択型・オムニバス形式）
* **全体進行**: 「ステージ（クエスト）選択型」のマップで進行。
* **画面遷移フロー**:
  * **「タイトル」 ──> 「ステージ選択・ショップ」 ──> 「バトル前演出（シナリオ導入）」 ──> 「バトル（救護ミッション）」 ──> 「ステージ選択・ショップ」** のループ構造。
* **シナリオ形式**: オムニバス形式（一話完結型）。ステージごとに異なる街、異なる傷病者、異なるシチュエーションが描かれ、様々な状態の救急・災害対応を学習します。
* **進行の流れ**: 
  1. タイトル画面から開始。
  2. ステージ選択画面（およびショップメニューへのアクセス）にて、挑戦するステージを選択。必要に応じて、所持しているゲーム内通貨で消耗品アイテムを買い足す。
  3. ステージを選択すると、バトル前演出（有事や傷病者の発生、災害の勃発シナリオ）が発生。
  4. 救護バトル（ミッション）へ移行。
  5. 適切な処置を行い、救護に成功（医師や救助組織への引き継ぎ完了）することでステージクリア。
  6. クリア時のバイタル状態などの結果に応じた報酬（ゲーム内通貨など）を獲得し、「ステージ選択・ショップ」画面に戻る。新たなステージやアイテム、スキルが解放される。

### 3.2 プレイヤーキャラクター
* **立場**: 特別な力（魔法や専門資格）を持たない「一般市民」。
* 旅先や日常の中で突如遭遇したトラブルに対して、身の回りにあるものや正しい知識を頼りに応急救護を行います。

## 4. バトル・ミッションシステム

### 4.1 コマンド選択式バトル
* **基本ルール**: 傷病者の状況を観察し、正しい救護手順をコマンドから選択する。
* **思考時間重視設計**: プレイヤーがじっくり判断できるよう、制限時間は「コマンド実行ごとに一定量減少する（ターン制のような仕組み）」とし、リアルタイムでの減少は行わない。
* **バイタルシステム（5段階のステータス）**:
  * 救助対象（傷病者）の容体は**1〜5の5段階ステータス**で視覚的に表現される。
  * ステージ開始時の初期ステータスは**「3」**。
  * **「5」**に到達すると容体が完全に安定し、救護完了（ステージクリア条件の達成）。
  * **「1」**の時にさらにダメージ（不適切な行動）を受けるとゲームオーバー（救護失敗）となる。

### 4.2 知識重視の成否判定とメッセージ
* **成功確率の扱い**: 正しいコマンド（手順として適切なもの）を選べば**100%成功**する。
* **変動システム**: 
  * 正しい行動を選択すると、バイタルステータスが**1段階上昇**する。
  * 不適切な行動を選択すると、バイタルステータスが**1段階減少**する。
* **バトルテキスト（メッセージ仕様）**:
  * 正しいコマンド選択時：**「正しい行動を選んだ」**
  * 容体（ステータス）上昇時：**「容体が良くなった」**
  * 誤ったコマンド選択時：**「不適切な行動を選んだ」**
  * 容体（ステータス）減少時：**「容体が悪化した」**

## 5. 成長・育成要素

プレイヤーキャラクターはステージクリア等で得られる経験によって成長します。

* **行動速度の上昇**: 各コマンド実行時の残り時間消費（タイムリミットの消費量）を抑える、またはボーナス猶予が獲得できる。
* **アイテムの解放・入手**: 応急処置で使用できる魔道具や備品が解放され、ショップで買い足すことが可能になる。
* **成功確率・判定の有利化**: 
  * 周囲の一般市民や、ゲームを進めると低確率で登場する「特殊な職業のNPC」に協力を仰いだ際、協力が成功する確率が上昇する。
  * 不適切な行動（誤ったコマンド）を選択してしまった際、致命的な失敗（容態悪化）にならずに未遂に留まる、またはペナルティが発生しない「失敗回避確率」が上昇する。
