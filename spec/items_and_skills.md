# 救急・救命・救助知識学習RPG 仕様書：魔道具（アイテム）＆成長システム

## 1. 魔道具（現実の救急・救助アイテムの翻訳）

世界観（ファンタジー）に合わせ、現実の救急道具を「古代の魔道具」や「冒険者の備品」として再定義します。これにより、ゲームとしてのエンタメ性を保ちつつ、直感的に現実のアイテムと結びつけられるようにします。

### 1.1 アイテム翻訳および詳細説明の露出ルール
世界観への没入感と実用的な学習効果を両立するため、**現実世界とのリンク（「これは現実の〇〇に相当します」といった実用解説）は以下の箇所にのみ限定して表示**します。
1. **アイテムの長押し画面**: バトル中や編成画面でアイテムアイコンを長押し（ロングタップ）した際に表示される詳細ポップアップ。
2. **図鑑（アーカイブ）メニュー**: ホーム画面からいつでもアクセスできる知識習得用データベース。

### 1.2 アイテム定義リスト

| 現実のアイテム | ゲーム内名称（魔道具など） | 長押し時・図鑑での学習用テキスト（現実解説） |
| :--- | :--- | :--- |

## 2. アイテム消費と買い足しシステム（ゲーム内通貨）

アイテムは一部の消耗品（包帯など）に「消費」の概念が存在します。

* **消耗品と常備品**:
  * **消耗品（例：エルフの癒やし布、聖水の滴）**: 使用するとステージ終了時に消費される。
  * **常備品（例：古代の蘇生魔石、息吹のベール）**: 一度解放・入手すれば、消費されることなく何度でもステージに持ち込める。
* **所持制限**:
  * 消耗品アイテムの最大所持数は、各種**「最大5個」**までに制限される。
* **獲得報酬（ゴールド）**:
  * ステージクリア時、クリア時の結果（最終バイタルステータス値など）に応じて、**3 〜 7 ゴールド** の間で変動する報酬を獲得する（例：最良クリアで7ゴールド、ギリギリで3ゴールドなど）。
* **買い足しシステムと購入価格**:
  * 獲得したゴールドを使用して、ホーム画面（ステージ選択画面など）の「ギルドショップ（道具屋）」から消耗品アイテムを買い足すことができる。
  * 消耗品アイテムの基準購入価格は、基本クリア報酬の中央値（5ゴールド）を2回得た金額、すなわち **10 ゴールド** を基本とする。

## 3. 成長システム

プレイヤーキャラクター（一般市民）はステージ攻略で得られる経験によって成長し、以下のステータスが向上します。

1. **行動速度（敏捷性 / クイックネス）**
   * コマンドを1回選択した際のタイムリミットの消費量が減る。
2. **失敗回避（冷静さ / マインドセット）**
   * 不適切なコマンドを選んでしまった際、エラー（傷病者の容態悪化）を回避し、「あ、これは違う！」と踏みとどまる確率。レベルアップによってこの確率が上昇する。
3. **周囲の協力（人徳 / カリスマ）**
   * 「応援要請」コマンドを使用した際、一般市民NPCが協力してくれる確率、および「特殊な職業のNPC（兵士や魔法使い）」が出現する確率（**初期値から最大15%以下**の範囲）が上昇する。
